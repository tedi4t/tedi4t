# List of projects

- [Tunasync](https://github.com/Scopics/tunasync) - async library for JS
- [Image belabour](https://github.com/Scopics/image-belabour) - project for parallel processing images by transform function with NodeJS
- [Intelligent cloud system](https://github.com/Intelligent-Cloud-System) - project to store files for organizations
- AGM - system to hold big meeting. We had one meeting with 5500 participants. This system uses AWS MediaLIVE to share video for nearly unlimited amount of people. It also uses telnyx to handle people who want to join meeting from their phones.
